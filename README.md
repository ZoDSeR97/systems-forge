# Systems Forge

Systems Forge is a workspace for designing clean, composable native system components with a stable C ABI boundary for cross-language interoperability.

## Repository layout

- [.gitignore](.gitignore)
- [apps/](apps/) — consumer applications
- [build/](build/) — build scripts and CI helpers
- [components/](components/) — reusable system components
- [docs/](docs/) — documentation
- [interfaces/](interfaces/) — shared ABI/interface definitions
- [playground/](playground/) — language examples and experiments

## Notes

This README is a high-level entry point. See the folders above for examples and concrete implementations.
